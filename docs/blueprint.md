# **App Name**: FitKit 2025

## Core Features:

- Gym & Studio Directory: Display dynamic listings of gyms and studios, searchable by city, category, and amenities, integrated with location APIs to allow proximity-based filtering.
- Membership & Pricing: Interactive display of subscription plans with animated transitions between pricing tiers, member benefits, and promotional offers.
- User Authentication: Secure registration/login with profile personalization and password recovery.
- Booking & Scheduling: End-to-end class/session booking integrated with calendar tools and real-time booking confirmation.
- My Bookings Dashboard: Centralized dashboard for managing upcoming and past bookings, schedule edits, cancellations, and attendance check-ins.
- Contact & Support: FAQs, contact form, and live support chat with real-time query submission.
- AI-Powered Gym Recommendations: Provide users with personalized gym or class suggestions, as determined by a tool using user data and gym information.

## Style Guidelines:

- Primary color: Vibrant Fitness Blue (#E63946), symbolizing strength and energy.
- Background color: Light Aqua Blue (#E0F7FA), clean and motivating.
- Accent color: Deep Charcoal (#212121) for CTAs, icons, and headers.
- Font: 'PT Sans' (sans-serif) for a modern, trustworthy aesthetic.
- Fitness-themed icons and SVG illustrations of gym activities.
- Rounded rectangular CTAs with hover animations.
- Hero section entry transitions using GSAP ScrollTrigger, subtle hover effects, and content reveal animations triggered by scroll depth.
